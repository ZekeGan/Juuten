## 使用技術

- `React` 前端框架
- `Redux` 狀態管理
- `React Router` 路由工具
- `styled-components` css-in-js
- `Redux-Thunk` 處理非同步狀態
- `Draftjs` 筆記文本編輯器
- `React-beautiful-dnd` dnd(拖移)組件
- `webpack` 打包整合工具
